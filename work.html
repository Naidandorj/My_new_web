Ажлын туршлага

Би олон жилийн хугацаанд програм хангамж болон техник хангамжийн чиглэлээр ажилласан. Доорх ажил эрхлэлтийн туршлагаас миний мэргэжлийн чадварууд тодорхой харагдана.

Ажилласан компаниуд:

  • 2010-2012: Source Code Co., Ltd - Техник хангамжийн инженер
  • 2013-2014: Зүүнхараа Хүүхдийн ордон - Компьютер, биеийн тамирын багш
  • 2015: Орхон аймгийн Хөдөлмөрийн хэлтэс - Програм хангамжийн мэргэжилтэн
